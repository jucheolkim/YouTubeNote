## 사용 방법

각 노트는 HTML 파일입니다. 보는 방법:

1. 위 표에서 노트 클릭
2. GitHub 화면에서 우측 상단 `Download raw file` 클릭
3. 다운받은 파일을 더블클릭하면 브라우저로 열림
4. 좌측 설명 + 우측 영상 임베드로 구성됨

## 폴더 구조

```
YouTubeNote/
├── README.md                    이 파일 (전체 목차)

└── (앞으로 추가될 폴더들)
```
