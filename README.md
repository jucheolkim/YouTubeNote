# YouTubeNote

유튜브 영상을 보고 정리한 개인 학습 노트 모음입니다.
영상 임베드 + 챕터별 설명 + 실습 체크리스트 형태로 구성되어 있습니다.

## 학습 노트 목록

| 분야 | 제목 | 원본 영상 | 정리일 | 바로가기 |
|---|---|---|---|---|
| 디자인/AI 도구 | 클로드 디자인 30분 풀코스 | [시현의 모험](https://www.youtube.com/watch?v=MNMlzq7shi0) | 2026-05 | [열기](https://jucheolkim.github.io/YouTubeNote/claude-design/클로드디자인_30분풀코스_교재.html) |

## 저작권 안내

본 저장소의 학습 노트는 모두 비영리 개인 학습 목적으로 작성되었습니다.

- 각 영상의 저작권은 해당 영상의 원작자에게 있습니다
- 학습 노트는 원본 영상의 시청을 대체하지 않으며, 영상의 가치를 보완하는 보조 자료입니다
- 노트 내용은 원본 영상을 보고 정리한 개인적 해석과 메모이며, 강사의 표현이나 견해를 그대로 옮기지 않으려 노력했습니다
- 원본 영상은 항상 출처 링크를 통해 직접 시청하실 수 있습니다

원작자께서 노트 삭제를 요청하시면 즉시 비공개 처리하겠습니다.

## 사용 방법

위 표의 "열기" 링크를 클릭하면 해당 학습 노트를 바로 볼 수 있습니다.
좌측에 챕터별 설명, 우측에 해당 영상 구간이 임베드되어 있어 함께 학습할 수 있습니다.
