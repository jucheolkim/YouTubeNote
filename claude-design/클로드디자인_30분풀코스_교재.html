클로드 디자인이란?
이게 뭐예요?
앤트로픽이 만든 디자인 전용 도구. 클로드 코드와 비슷하게 자연어로 시키면 디자인을 만들어 줍니다. 모델은 Opus 4.7이 붙어 있어요.
왜 중요해요?
디자인 비전문가도 시각적인 결과물을 만들 수 있고, 디자인이 어떻게 돌아가는지 시각적으로 배울 수 있는 교보재 역할을 합니다. 피그마의 시가총액이 줄었다는 뉴스가 나올 정도로 업계 변화가 큽니다.
"AI 슬롭" 걱정은요?
강사 의견: 디자이너가 아닌 사람한테는 어차피 AI 슬롭이 더 낫습니다. 너무 신경 쓰지 말고 만들어 보세요.

무료로 가능?
아니요. 유료 플랜이 필요합니다.
어떤 플랜?

사용량은 어디서 봐요?

주의 · 페이지 하나 만드는데 사용량 10~20%가 그냥 사라집니다. $20 플랜은 금방 동납니다. 토큰 소모가 큰 도구라는 점을 미리 알고 시작하세요.

디자인 시스템이 왜 필요한가
디자인 시스템이 뭐예요?
"이 브랜드는 이런 규칙을 따른다"는 시각적 가이드북. 폰트 크기, 색상, 버튼 모양, 카드 디자인, 라운드 정도 등을 미리 정해 놓는 문서입니다.
왜 만들어요?
규칙을 미리 정하면 일관된 디자인이 나옵니다. AI에게 시킬 때도, 사람이 직접 작업할 때도 마찬가지. 프로 디자이너는 이걸 만들고 시작하는 게 기본입니다.
초보자에게 의미는?
디자인 시스템이라는 문서가 있다는 사실 자체를 모르는 게 보통인데, 클로드 디자인이 이걸 자동으로 만들어 주면서 디자이너의 역할 범위를 자연스럽게 알게 됩니다.
디자인 시스템 만드는 입력값
✓
컴퍼니 이름 / 프로젝트 이름 입력
✓
(있으면) GitHub 주소 또는 코드 업로드
✓
(있으면) 피그마 파일 업로드
✓
"만들기" 버튼 클릭 → UI 키트 / 타이포 / 컬러 자동 생성
강사 메모 · 자동 생성된 디자인 시스템을 그대로 쓰진 않고, 참조용으로 보고 있다고 합니다. 초안을 받고 마음에 드는 부분만 차용하는 식.

결과물 평가 + 코멘트로 수정
첫 결과물은 70~80점. 여기서부터는 깎아 나가는 작업입니다.
수정하는 두 가지 방법
① 직접 편집: Edit 모드에서 텍스트나 요소를 클릭해 폰트/사이즈 등 직접 수정.
② 코멘트로 지시: 특정 섹션을 클릭하고 "이 섹션의 영어를 한글로 바꿔줘" 같이 코멘트를 남기면 그 섹션에 한정해서 수정됩니다.
업계 용어가 필요한 이유 · "내비바가 두껍다 / 히어로 CTA 간격이 좁다 / 갭을 늘려라 / 스탯 카드가 모바일에서 안 보인다" 같은 정확한 요청을 하려면 용어를 알아야 합니다. AI가 뱉는 텍스트를 그때그때 검색하면서 단어집을 만들어 가는 게 좋은 학습 방법.
